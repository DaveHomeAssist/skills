# PromptLab Skills Reference

Skill definitions for the PromptLab ecosystem. Each skill is a reusable prompt pattern with a defined purpose and output contract.

---

## Core Skills

| Skill | Purpose | Description |
|---|---|---|
| **Extract** | Systematically pull structure from unstructured content | Extract tasks, decisions, facts, ideas, entities, dates from messy notes. |
| **Format** | Enhance readability and Notion formatting | Reformat content into clear headings, lists, and hierarchy; ensure Notion-ready structure. |
| **Summarize** | Compress content while preserving meaning | Produce concise narrative summaries, not just bullets. |
| **Transform** | Convert input to a different useful format | E.g., turn notes into blog posts, meetings into task lists. |
| **Classify** | Assign tags and categories | Categorize content by type, priority, intent; power database organization. |

---

## Cowork Skills

| Skill | Purpose | Description |
|---|---|---|
| **Critique** | Evaluate quality and identify weaknesses | Highlight strengths, weaknesses, and missing elements with improvement suggestions. |
| **Gap Analysis** | Find missing components or risks | Spot blind spots and suggest additions to plans or prompts. |
| **Decision** | Turn ambiguity into clear choices | List options, trade-offs, recommend a path, and explain why. |
| **System Builder** | Design structured workflows | Define components, inputs/outputs, dependencies for systems or prompts. |
| **Debug** | Diagnose problems and propose fixes | Identify root causes, provide evidence, outline steps to resolve and prevent issues. |

---

## Notion-Specific Skills

| Skill | Purpose | Description |
|---|---|---|
| **Database Builder** | Create Notion DB schema | Define properties, types, and example rows for Notion databases. |
| **Prompt Refactor** | Refine prompts into reusable skills | Clean up instructions, tighten rules, define output contracts. |
| **Knowledge Graph** | Map entities and their relationships | Extract entities and link them to build a second-brain knowledge graph. |
| **Workflow Automation** | Define automated triggers and actions | Specify triggers, actions, tools, and output formats for automation. |

---

## Advanced Skills

| Skill | Purpose | Description |
|---|---|---|
| **Intent Tagging** | Decompose user intent into atomic units | Identify intent units, assign priority and ambiguity level. |
| **Prompt Decomposition** | Split complex tasks into steps | Create step-by-step execution plans with sub-prompts. |
| **Synthesis** | Combine multiple inputs into insights | Detect patterns, draw conclusions, and make recommendations. |
